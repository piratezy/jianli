Docusaurus的相关知识

2023年2月25日 · 阅读需 2 分钟
朱岩
前端小菜鸡

想要创建一个blog，首先就要知道怎么来创建，那么第一就是要选择出一个合适的框架。 我这里用的就是docusaurus框架，这个框架是facebook开发的一款基于react的框架，我觉得很喜欢。 那么接下来讲一下怎么开发这个框架：

安装

首先，在命令行输出

npx create-docusaurus@latest my-website classic

设置config.js

  url: 'https://piratezy.github.io/',//表示你github仓库的网址
  baseUrl: '/',
  organizationName: 'piratezy',//作者github的名字 
  projectName: 'piratezy.github.io',//仓库名字

启动项目

在命令行输入

yarn start

构建项目

在命令行输入

yarn build

配置项目

上传代码时，有2个注意事项： deploymentBranch来指定分支！而分支表示的是你github上的关联账号的分支

deploymentBranch:"gh-pages",

其次还要在系统上配置环境变量:

GIT_USER:git用户名

接着在命令行输入：

yarn deploy

接下来就都弄好了
